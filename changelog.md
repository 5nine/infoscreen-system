# Changelog

Alla viktiga ändringar i detta projekt kommer att dokumenteras i denna fil.

## [1.0.0] - 2023-12-16
### Added
- Första släppet av Själevads Bygg Info-Skärm System
- Komplett info-skärm med 1920x1080 upplösning
- Touch kontrollpanel för 800x1280 (portrait)
- Admin panel för bildhantering
- Auto-update system med GitHub integration
- Väderintegration för Örnsköldsvik
- Kalenderintegration med Google Calendar
- Systemd service för automatisk start
- Backup och restore funktionalitet
- Diagnostikverktyg
- Fullständig dokumentation

### Technical
- Node.js Express backend
- WebSocket för realtidskommunikation
- ImageMagick för bildbehandling
- Auto-genererade thumbnails
- Responsiv design
- Touch-optimized gränssnitt
